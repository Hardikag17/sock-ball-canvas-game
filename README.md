# `Sock Ball Game`
Game in canvas with socket.js to show ball between multiple computers

# steps

**webi-ui**

- cd `web-ui`
- run `local server` or `python -m http.server 80`

  
** `isDevMode = false`

**server** (*Not required*)

- cd `server-express`
- run `npm i`
- run `nodemon init`

  
** `isDevMode = true` & `websockTargetHost: localhost`
